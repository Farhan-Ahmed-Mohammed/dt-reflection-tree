# Daily Reflection Tree – DT Fellowship Assignment

This project implements a **deterministic reflection agent** using a structured decision tree.  
The system guides a user through an end-of-day reflection across three psychological axes without using any LLM at runtime.

---

## 🧠 Core Idea

The reflection tool is built as a **decision tree**, where:
- Each question has fixed options
- Each option leads to a predefined branch
- Reflections are pre-written and deterministic

This ensures the system is:
- Predictable
- Auditable
- Free from AI hallucinations

---

## 🔍 Three Axes of Reflection

### 1. Locus of Control
Identifies whether the user attributes outcomes to:
- Internal factors (effort, decisions)
- External factors (luck, others)

---

### 2. Contribution vs Entitlement
Evaluates whether the user:
- Contributed beyond expectations
- Stayed neutral
- Expected more than they gave

---

### 3. Radius of Concern
Expands perspective from:
- Self → Team → Customers
